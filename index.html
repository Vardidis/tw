📖 Οδηγίες Χρήσης

1

Copy το Script URL

Αυτό είναι το hosted script. Copy το παρακάτω κώδικα:

javascript:$.getScript('Loading...');
2

Δημιούργησε Bookmark

Δημιούργησε νέο bookmark στον browser σου:

Name: Attack Tracker
URL: Paste το παραπάνω code

Tip: Μπορείς να το βάλεις στη bookmarks toolbar για εύκολη πρόσβαση!

3

Single-Click Χρήση

Όταν παίζεις Tribal Wars:

1 κλικ: Κάνε κλικ στο bookmark "Attack Tracker"

✅ Αν είσαι στις εισερχόμενες: Ξεκινάει listening αμέσως!
✅ Αν είσαι σε άλλη σελίδα: Πηγαίνει στις εισερχόμενες και ξεκινάει listening!

🎯 Panel εμφανίζεται με όλες τις επιθέσεις
🎯 Live countdown & unit detection
🎯 Αυτόματη καταγραφή νέων επιθέσεων
4

Ρύθμιση World Speed

Πρώτη φορά που τρέχει το script, ρύθμισε τα settings:

World Speed: Βρες στο Settings του game (π.χ. 1.0, 2.0, κλπ.)
Unit Speed: Συνήθως 1.0 (εκτός αν ο κόσμος έχει special settings)

Το panel θα εμφανιστεί δεξιά με τα configuration fields
